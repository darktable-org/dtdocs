AI masking tools require AI features to be enabled and a model for the _mask_ task active in [AI preferences](../../../preferences-settings/ai.md) – see the [AI features overview](../../../special-topics/ai/overview.md) for 1details.

Currently there is one AI masking tool implemented called __object__. It allows for an object to be clicked and the mask refined with further clicks. The result is then stored as a regular path shape which can be edited further, similar to other drawn shapes. See [drawn masks: object](../drawn)
